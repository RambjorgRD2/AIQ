<task type="checkpoint:human-action" gate="blocking">
  <name>Task 1: Add custom domain in Vercel and set DNS records</name>
  <what-to-do>
**Step 1 — Add domain to Vercel project:**
1. Vercel Dashboard → Project → Settings → Domains
2. Enter `www.theaiq.org` → Add
3. Vercel will show the required DNS record: CNAME `www` → `cname.vercel-dns-0.com`
4. Also add `theaiq.org` (apex) if you want the apex to redirect to www

**Step 2 — Add apex redirect (recommended):**
1. Vercel Dashboard → Settings → Domains → Add `theaiq.org`
2. Vercel will offer to set it as a redirect to `www.theaiq.org` — accept this
3. This requires an A record: `@` (apex) → `76.76.21.21`

**Step 3 — Add DNS records at your registrar:**

Log into the DNS control panel for theaiq.org (Cloudflare, Namecheap, GoDaddy, or similar).

Add these records:

| Type  | Name | Value                    | TTL  |
|-------|------|--------------------------|------|
| CNAME | www  | cname.vercel-dns-0.com   | Auto |
| A     | @    | 76.76.21.21              | Auto |

If the registrar uses Cloudflare proxying (orange cloud), set it to DNS-only (grey cloud) for the CNAME record — Vercel handles SSL directly.

**Step 4 — Wait for DNS propagation:**
CNAME changes typically propagate in 5-30 minutes. Check progress:
```bash
dig www.theaiq.org CNAME +short
# Should return: cname.vercel-dns-0.com.
```

Or use https://dnschecker.org to verify global propagation.

**Step 5 — Verify Vercel shows domain as "Valid":**
Vercel Dashboard → Settings → Domains → www.theaiq.org should show a green checkmark and "Valid" status with SSL certificate.
